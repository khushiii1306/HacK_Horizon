## 🔧 Customization Quick Guide

### Change Colors
Edit `styles` constant in `src/App.jsx`:
```javascript
.btn-primary { background: #YOUR_COLOR; }
```

### Update Company Data
Modify near top of `src/App.jsx`:
```javascript
const LISTINGS = [ /* your data */ ];
const VOLUNTEERS = [ /* your data */ ];
const PROVIDERS = [ /* your data */ ];
```

### Add Navigation Items
Update the sidebar arrays in dashboard components

### Change Content
Search for text in `src/App.jsx` and replace
